🧠 Prompt de Instrução – Importação CoinPoker (.CSV)
Você deve implementar um interpretador de arquivos .CSV da CoinPoker que respeite as seguintes regras para importar corretamente os torneios para a plataforma.

📥 Estrutura do Arquivo
Cada linha representa uma transação e possui os seguintes campos relevantes:

Description: Contém o nome do torneio após o caractere ?

Amount: Valor da transação, negativo (buy-in), zero (eliminação sem prêmio) ou positivo (prêmio)

Date: Exemplo: 2024-05-15 00:24:53 GMT-3 | 368270688

🔍 Regras de Processamento
1. Filtragem Inicial

Ignore qualquer linha em que Description não comece com "NL Hold'em"

2.Identificação de Torneios

Use a parte da Description após o caractere ? como identificador do torneio

Use apenas a parte antes do  horario em Date, convertida de YYYY-MM-DD para DD-MM-YYYY

3.Agrupamento por Torneio


Um torneio válido deve conter:

Uma linha com Amount negativo (entrada)

Uma linha com Amount igual a 0 ou positivo (eliminação ou premiação)

Ambas as linhas devem ter o mesmo identificador de torneio e mesma data

4.Montagem do Registro Final
Para cada torneio identificado:

Nome do Torneio: texto após o ? em Description

Data: data formatada para DD-MM-YYYY

Buy-in: valor absoluto do Amount negativo

Prize: valor do Amount da linha correspondente (0 ou positivo)

Posição: null (não disponível)

Site: "CoinPoker"

Rake: 0

5.Conversão de Moeda

Os valores estão em USDT, que devem ser tratados como USD (sem conversão)

⚠️ Observações Adicionais
Um torneio só deve ser importado se contiver exatamente uma linha de entrada (buy-in) e uma linha de encerramento (prêmio ou eliminação).

Se o mesmo nome de torneio aparecer mais de duas vezes na mesma data, considere que são instâncias diferentes do mesmo torneio, e trate como torneios separados.

Nunca agrupe ou some prêmios de linhas diferentes para o mesmo identificador. Um par de linhas = 1 torneio.
