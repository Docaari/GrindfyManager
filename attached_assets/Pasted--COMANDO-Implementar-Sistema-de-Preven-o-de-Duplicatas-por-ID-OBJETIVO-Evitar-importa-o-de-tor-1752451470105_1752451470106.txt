🔄 COMANDO: Implementar Sistema de Prevenção de Duplicatas por ID
OBJETIVO: Evitar importação de torneios já existentes usando Tournament ID
IMPLEMENTAR:
1. VERIFICAR SISTEMA ATUAL DE DUPLICATAS:

Console.log para ver se já existe alguma verificação de duplicatas
Identificar se está usando outros campos (nome + data + buyin)
Ver se Tournament ID já está sendo usado na verificação

2. IMPLEMENTAR VERIFICAÇÃO POR TOURNAMENT ID:

Antes de processar cada linha do CSV, verificar se Tournament ID já existe
Query no banco: SELECT id FROM tournaments WHERE tournamentId = ? AND userId = ?
Se existir: Pular linha e contar como "duplicado ignorado"
Se não existir: Processar normalmente

3. LÓGICA SUGERIDA PARA O BACKEND:
Para cada linha do CSV:
  1. Extrair Tournament ID da linha
  2. Verificar se ID já existe no banco para o usuário
  3. SE existe: adicionar a lista de "duplicados ignorados"
  4. SE não existe: processar e adicionar à lista de "novos torneios"
4. IMPLEMENTAR PARA TODOS OS SITES:

PartyPoker: Usar campo "Game ID" como Tournament ID
PokerStars: Usar campo específico de ID do torneio
WPN: Identificar campo correto de ID
Outros sites: Mapear campo de ID específico de cada site

5. MELHORAR FEEDBACK PARA O USUÁRIO:

Duplicados ignorados: "X torneios já existiam e foram ignorados"
Detalhamento: Mostrar quais IDs foram ignorados
Estatísticas: "15 novos, 3 duplicados, 0 erros"

6. OTIMIZAR PERFORMANCE:

Batch verification: Verificar múltiplos IDs de uma vez
Cache temporário: Durante importação, manter lista de IDs já verificados
Query otimizada: Usar WHERE IN para verificar múltiplos IDs

7. TRATAMENTO DE EDGE CASES:

ID vazio/nulo: Usar sistema de duplicatas atual (nome+data+buyin)
ID inválido: Processar mas alertar no log
Formatos diferentes: Padronizar ID antes de verificar

8. TESTE VALIDAÇÃO:

Teste 1: Importar arquivo novo → todos devem ser processados
Teste 2: Importar mesmo arquivo → todos devem ser ignorados como duplicados
Teste 3: Importar arquivo com mix → separar novos de duplicados

RESULTADO ESPERADO:

Sistema eficiente de prevenção de duplicatas
Feedback claro sobre duplicados ignorados
Performance otimizada para importações grandes
Funciona com todos os sites suportados
