ETAPA 3: SISTEMA DE CRIAÇÃO DE SESSÃO
IMPLEMENTAR CRIAÇÃO DE SESSÃO:
1. API POST /api/grind-sessions:
   - Receber dados: userId, data, torneios planejados
   - Salvar sessão com userPlatformId correto
   - Gerar ID único para a sessão
   - Status inicial: "active" ou "planning"

   - Timestamps de criação e última atualização

3. Validações:
   - Verificar se usuário não tem sessão ativa
   - Validar se há torneios planejados para o dia
   - Confirmar permissões de grind_session_access

ETAPA 4: ISOLAMENTO POR USUÁRIO
GARANTIR ISOLAMENTO COMPLETO:

2. Verificações de segurança:
   - Usuário só vê suas próprias sessões
   - Não pode acessar sessões de outros usuários
   - Dados não vazam entre contas

3. Logs de auditoria:
   - Log de criação de sessão por usuário
