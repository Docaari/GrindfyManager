# ETAPA 4: Email de Reset de Senha Baseado no Email de Confirmação

## 🎯 Objetivo
Criar o **Email de Reset de Senha** usando como base o design e estrutura do Email de Confirmação já reformado.

## 🎨 Design Base
**Usar exatamente o mesmo layout do Email de Confirmação:**
- **Mesmo header** com logo + "Grindfy" (Grind branco + fy verde)
- **Mesmo fundo escuro** no header
- **Mesma estrutura** de cards e seções
- **Mesmas cores** e tipografia
- **Mesmo footer** com informações da plataforma

## 📄 Conteúdo do Email de Reset de Senha

### Título Principal:
```
🔒 Reset de senha
```

### Mensagem Principal:
```
Recebemos uma solicitação para redefinir a senha da sua conta no Grindfy.

Clique no botão abaixo para criar uma nova senha:
```

### Botão Principal:
- **Texto:** "🔐 Redefinir Senha"
- **Cor:** Verde da plataforma (#00ff88)
- **Tamanho:** Botão grande e destacado (mesmo estilo do Email de Confirmação)

### Aviso de Expiração:
```
⚠️ Este link expira em 1 hora

Por segurança, este link de reset só é válido por 60 minutos.
```

### Informação de Segurança:
```
Se você não solicitou este reset, pode ignorar este email com segurança. Sua senha não será alterada.
```

### Footer:
```
Grindfy
© 2025 Grindfy - Plataforma de Analytics para Poker
Este email foi enviado automaticamente, não responda.
```

## 🔧 Implementação
- **Copiar estrutura** do Email de Confirmação
- **Alterar apenas o conteúdo** conforme especificado
- **Manter design** e cores idênticos
- **Usar mesmo botão** (mudando texto e link)

## ✅ Resultado Esperado
- **Visual idêntico** ao Email de Confirmação
- **Conteúdo focado** em segurança
- **Aviso claro** sobre expiração (1 hora)
- **CTA direto** para redefinir senha
- **Consistência total** entre os emails

**Foco:** Fornecer acesso seguro para redefinição de senha com informações claras sobre validade e segurança.
